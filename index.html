Projects

I designed an electric spirit level using the MSP432 For my embedded systems and programming course, I used the MS432 development board and an accelerometer to make an electronic spirit level, which could tell the level of straightness of a surface with very little error. A battery powered spirit level that fits in a pocket in my opinion would be very handy and useful. It is economic and simply takes less space than the mechanical ones we are all used to. The code can be found here in my github repo.

Wrote a finite element analysis algorithm. Finite element analysis helps us to model structures and phenomenon that might be too expensive or simply impossible to test out. I wrote, the algorithm which was used in testing the effects on applying different forces on different structures including the eiffel tower.

The idea behind the project was to be able to allow a microcontroller make decisions such as informing home occupants or controlling servos to close windows when there was rainfall. Temperature and humidity readings were collected over a period using the DHT22 sensor and an ESP8266 microcontroller, sent to the thingspeak platform, cleaned up and a model was trained (with an f1_score of 96.6), and deployed on an ESP8266 wemos D1 microcontroller. The code can be found here in my github repo.
