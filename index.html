Experimental Platform for Advanced Control of a Two-Wheel Self-Balancing Robot This project investigates the implementation and experimental evaluation of modern control algorithms on the M5Stack Bala2 Fire self-balancing robot. Rather than treating the robot as a finished product, it serves as a research platform for studying balance control, state estimation, system identification, and real-time embedded control.

The platform has been used to implement and compare multiple control strategies, including classical PID control, Linear Quadratic Regulator (LQR), and Data-enabled Predictive Control (DeePC). Current work focuses on improving model accuracy through system identification and investigating the transition from model-based control to data-driven control methods.

The project combines embedded C++, IMU sensor fusion, motor control, real-time estimation, and experimental validation, providing a practical framework for developing advanced control algorithms on embedded robotic systems.

Technologies: Embedded C++, M5Stack Bala2 Fire, IMU Sensor Fusion, PID, LQR, DeePC, System Identification, MATLAB, Control Systems.

Autonomous Package Delivery Robot with Active Load Stabilization

Developed as part of the Applied Systems Engineering course, this project involved the design and implementation of an autonomous mobile robot capable of collecting and delivering packages while maintaining stability over uneven terrain.

The challenge required the robot to autonomously navigate a predefined environment, identify package locations using color markers, pick up packages, transport them to their corresponding delivery locations, and overcome obstacles while maintaining a stable payload.

The robot was developed as a complete mechatronic system combining mechanical design, embedded programming, sensing, and control. The mechanical structure was designed using CAD, while the software architecture implemented autonomous navigation, line following, package detection, manipulation, and task sequencing.

A major challenge was crossing elevated obstacles while keeping a ping-pong ball balanced on the robot platform. An IMU-based gimbal stabilization system was implemented to compensate for robot tilt and actively maintain a horizontal platform.

Autonomous System Components:
  • Line following using a sensor array
  • Color-based package identification
  • Finite state machine architecture for mission execution
  • Motor control using encoder feedback
  • Servo-based package pickup and delivery mechanism
  • IMU-based gimbal stabilization control

The project required collaboration across mechanical design, electronics, and software development. My contributions focused on embedded programming, robot control, sensor integration, and autonomous behavior implementation.

Technologies: Embedded C++, STM32 Nucleo, Mbed OS, DC Motors, Encoders, IMU Sensor Fusion, Line Following, Color Detection, Servo Control, CAD Design, Autonomous Robotics.
